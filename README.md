## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `src/app/page.tsx`. The page auto-updates as you edit the file.

## SEO configuration

Set `SITE_URL` in your environment before production builds so canonical URLs, the sitemap, robots metadata, and social preview metadata all point to the real domain.

- Shared metadata config lives in `src/lib/site.ts`.
- Root metadata and site-wide JSON-LD live in `src/app/layout.tsx`.
- Homepage SEO and homepage structured data live in `src/app/page.tsx`.
- Practice route SEO is generated in `src/app/practice/[group]/page.tsx`.
- Crawl metadata is generated by `src/app/robots.ts` and `src/app/sitemap.ts`.
- Social preview art is generated by `src/app/opengraph-image.tsx`.
